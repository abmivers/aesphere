Operaciones

A través de esta pantalla podremos acceder a cada una de las fases que se realizan en las rondas del algoritmo AES facilitando así su comprensión y pudiendo comprobar en detalle como funciona este algoritmo de cifrado. En cada una de estas fases se opera con una matriz de 4x4 bytes llamada state, y es de esa forma (como matriz) como se introducirán los datos en la aplicación.

Con el botón "Cancelar" volveremos a la pantalla principal de la aplicación

Para una información más detallada y teórica de este proceso consulte los siguientes términos:
Cifrado,Descifrado, AddRoundKey, MixColumns, ShiftRows, SubBytes.
